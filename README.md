# Hobby Projects

A collection of small projects built to explore different technologies, concepts, and tools.
These projects range from quick experiments to more structured mini-applications.

---

## 📁 Project Structure

```text
Hobby-Projects/
├── Auto-Clicker/        # Simple automation tool for repetitive mouse actions
├── Chat-CLI/            # Command-line chat interface
├── File-Parser/         # Parses and processes structured/unstructured files
├── Local-Web-Server/    # Lightweight local server for testing APIs or static content
├── UI/                  # Basic UI/UX experiments and components
├── Web-Scraper/         # Scripts for extracting data from websites
```

---

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/archersnell/Hobbie-Projects.git
cd Hobbie-Projects
```

### Run a project

Each project is self-contained. Navigate into the desired folder:

```bash
cd Project-Name
```

Follow the instructions in that project's directory (if provided).
